## 📌 프로젝트 소개
이 프로젝트는 **코드팩토리의 플러터 프로그래밍 도서를 공부하며 구현한 flutter 앱**입니다. 주요 기능은 다음과 같습니다:
- 할 일 추가, 삭제, 수정
- 상태 변경 (예: 완료, 미완료)
- 검색 기능 및 필터

---

## 🚀 주요 기능
- **간단한 UI**: 사용자가 쉽게 할 일을 추가하고 관리할 수 있습니다.
- **반응형 디자인**: 모바일과 데스크톱에서 최적화.
- **백엔드 API 연동**: 데이터를 서버에 저장 및 관리.
